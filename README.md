# gushi-password-dicts
常用古诗句密码字典，用于安全研究。

## 介绍
一些自作聪明的人会将古诗混入自己的密码中，以对抗弱密码猜解，这个项目提供了碰运气的可能。  
建议结合其他已有信息，制作针对性更强的字典。  

## 这里有什么
数据来自[古诗文网](https://www.gushiwen.cn)，接口是抓移动端app来的。
